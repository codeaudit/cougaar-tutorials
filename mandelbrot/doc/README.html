Cougaar "Mandelbrot" Demo

This module contains the Cougaar "Mandelbrot" demo.

This demo illustrates various design solutions for implementing a "Mandelbrot Set" fractal image generator within Cougaar.

The point is to produce the exact same UI image but with very different backend implementations. The different design "versions" vary in their use of service APIs, blackboard objects, relays, distributed processing, etc.

Please see the accompanying "Cougaar Mandelbrot" slides:
  http://cougaar.org/docman/view.php/17/201/cougaar_mandelbrot.ppt

Installation:

Please follow the same installation steps as in the "Hello World" example, substituting "ping" for "hello".

The "Hello World" README is available online at:
   http://cougaar.org/cgi-bin/viewcvs.cgi/*checkout*/hello/doc/README.html?rev=1.1.1.1&cvsroot=tutorials#installation
and in the "cougaar-hello.zip" produced by the nightly build:
   http://build.cougaar.org/auto/HEAD/latest_success/

Usage:

If you lack a lib/mandelbrot.jar, you must first run:
   ant dist

The minimal "version 0" non-Cougaar demo can be run from the command line:
   ant test-popup

To run the "version 1" Cougaar demo, do:

  export COUGAAR_INSTALL_PATH=your Cougaar install path, which contains "lib/core.jar"

  export COUGAAR_SOCIETY_PATH=the mandelbrot base directory, which contains this "doc/README.html"

  export COUGAAR_RUNTIME_PATH=$COUGAAR_SOCIETY_PATH/run

  cd $COUGAAR_RUNTIME_PATH

  $COUGAAR_INSTALL_PATH/bin/cougaar ../configs/MandelbrotSociety_v1.xml MandelbrotRuntime_v1.xml
Then point a browser at:
   http://localhost:8800/mandelbrot
You should see a Mandelbrot Set image and HTML Form navigation options. Click on upper-left and lower-right points in the image to zoom in. To run the other "version X" Cougaar demos, e.g. "version 2", substitute the version number for the above "v1".
