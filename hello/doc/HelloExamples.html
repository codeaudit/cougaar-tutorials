Cougaar "Hello" Examples

This module contains several examples that show various ways of running the Hello World tutorials.

This module assumes that you have read the Hello World README and have successfully set up and run the demo. All examples are expected to be run in eclipse, but they can also be run as ant targets or from the command line. The following of the following sections describes how to run each example and how to interpret the output.

Index:

  • Hello World
  • Hello Logging
  • Hello Life Cycle
  • Hello Publication / Subscription
  • Hello Classic Subscriptions
  • Hello Query
  • Hello Timer
  • Hello Service
  • Hello Debug

Hello World:

  • The source code:
    src/org/cougaar/demo/hello/HelloPlugin.java

  • The society configuration file:
    configs/HelloWorldSociety.xml

  • Eclipse run command: 
     helloWorld

  • The Ant script target: 
     ant helloWorld

  • Unix command line (bash): 
      cd [HelloProjectDirectory]
  export COUGAAR_INSTALL_PATH=`pwd`/../../trimcougaar/TrimCougaar
  export CIP=$COUGAAR_INSTALL_PATH
  export COUGAAR_SOCIETY_PATH=`pwd`
  export COUGAAR_RUNTIME_PATH=`pwd`/run
  cd $COUGAAR_RUNTIME_PATH
  $CIP/bin/cougaar --runtime SingleNodeRuntime.xml --society ../configs/HelloWorldSociety.xml
