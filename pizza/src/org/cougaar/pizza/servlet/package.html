This package defines servlets for use with the Pizza Party application.

The HistoryServlet is a generic servlet for monitoring changes to the blackboard. Use the HistorySevletFriendly interface to allow the servlet to show meaningful information about your blackboard objects.

The PizzaPreferenceServlet is used to display who has been invited to the party and what kinds of pizza they want. It is the main UI for the pizza party app.
