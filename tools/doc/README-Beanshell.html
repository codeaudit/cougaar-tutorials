Bean Shell

Bean Shell provides an interactive 'shell' through which you can interact with a running Java program, by invoking methods on objects.

To use Bean Shell, include the following in your society xml: 


  <component class="org.cougaar.tutorials.bsh.BeanShellPlugin"/>

The interactive variable component will be bound to the plugin itself. The interactive variable sb will be bound to its ServiceBroker.

Example interaction (cut-and-pasted from a BeanShell window): 

    bsh % import org.cougaar.core.node.NodeIdentificationService;
    bsh % nis = sb.getService(component, NodeIdentificationService.class, null);
    <org.cougaar.core.agent.AgentManager$3@8c8d17>
    bsh % nis.getMessageAddress();
    <Node1>
